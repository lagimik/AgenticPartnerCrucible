# Microsoft Foundry Labs

Microsoft Foundry Labs is a Microsoft Foundry experience for making frontier AI experiments discoverable, interactive, and usable by builders and researchers.

## Positioning

Foundry Labs is framed as a bridge from Microsoft research to real-world use: users can discover frontier experiments, try them hands-on, understand the technical details, access related code or papers, and connect with the surrounding community.

## Notable Patterns

- "Breakthrough AI you can try today" homepage framing.
- Innovations catalog with 50+ experiments and filtering by domain, recency, or artifact type.
- Experiment pages with playgrounds, tags, technical explanations, GitHub/research links, and production/customer evidence.
- Community layer spanning events, blogs, publications, Discord, Reddit, Microsoft Research Blog, and Tech Community.
