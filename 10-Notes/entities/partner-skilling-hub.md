# Partner Skilling Hub

The Microsoft Partner Skilling Hub (skilling-hub.com) is the official Microsoft Partner training system. It provides exclusive training, certifications, and resources designed for Microsoft Partners.

## Key Features

- Official Microsoft Partner training platform
- Sponsored by Microsoft
- 10+ language support (en, fr, es, de, pt, it, ja, ko, zh-Hans, zh-Hant)
- AI-powered Skilling Agent for personalized guidance
- Microskilling content type (bite-sized learning units)
- Integration with Microsoft AI Cloud Partner Program (MAICPP)
- Operated by Digital Arts & Sciences (DAS)

## Content Types

- Microskillings (smallest atomic learning units)
- Courses
- Certifications
- Events and programs (including in-person)

## Skilling Agent

An AI-powered conversational assistant embedded in the platform that helps partners:
- Navigate skilling offerings
- Identify learning opportunities
- Build personalized skilling plans

## Relationship to LevelUp

LevelUp (TechnoFocus) is a complementary, Microsoft-sponsored platform focused on hands-on certification prep and deal readiness. Both serve the Microsoft partner skilling ecosystem.
