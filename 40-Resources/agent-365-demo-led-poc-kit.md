# Agent 365 Demo-Led PoC Kit

Source: https://aka.ms/Agent365PoC/Download

## Summary

Partner-facing demo-led proof-of-concept kit for starting Agent 365 customer conversations. The kit is positioned for engagements where partners help customers gain visibility into their agent estate, address governance challenges, or develop an AI agent strategy.

## Engagement Use

- Discover and discuss a customer's current or emerging AI agent estate.
- Frame governance challenges around visibility, control, and strategy.
- Use a repeatable engagement model to move from exploratory Agent 365 conversations into a structured PoC.
