# New Copilot Studio Tech Guide

A hands-on technical guide from the Microsoft CAT team covering the rebuilt Copilot Studio modern experience — now designed for complex, multi-step work with coordinated multi-agent systems.

## Key Topics

- **Connected Agents** — Parent agents delegating to specialized child agents
- **MCP Servers** — Model Context Protocol integration in Copilot Studio
- **Runtime Skills** — Per-agent skills that generate and execute Python at runtime
- **Generated Files** — Agents producing real file outputs
- **Workflows** — New workflow orchestration alongside agents

## Reference Scenario: BlastBox Omega

A complete runnable reference (retro-future game store) demonstrating:
- Parent agent orchestration
- Multiple connected agents
- Multiple MCP servers
- Runtime-generated Python execution
- File generation and return
