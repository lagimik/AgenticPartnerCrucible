# Partner Skilling Hub - Skilling Agent

AI-powered conversational assistant embedded in the Microsoft Partner Skilling Hub to help partners navigate skilling offerings and build personalized learning plans.

## Summary

The Skilling Agent demonstrates the pattern of embedding AI assistants directly into partner-facing platforms. It provides chat-based navigation of the skilling catalog, personalized recommendations, and learning path construction.

## Capabilities

- Chat-based conversational interface
- Session-based interaction
- Partner skilling catalog navigation
- Personalized learning plan generation
- Feedback collection (thumbs up/down)

## Pattern: AI Assistant Embedded in Workflow

This is an example of Microsoft deploying an AI agent within a partner-facing training platform. Rather than requiring partners to search manually, the agent acts as a guide through the skilling ecosystem.
