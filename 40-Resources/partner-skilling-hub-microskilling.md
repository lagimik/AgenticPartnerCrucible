# Partner Skilling Hub - Microskilling

Bite-sized learning modules within the Microsoft Partner Skilling Hub, representing the smallest atomic unit of skill-building content.

## Summary

Microskillings are a content type on the Microsoft Partner Skilling Hub (skilling-hub.com). They provide focused, rapid skill acquisition for Microsoft partners. The format sits below full courses and learning paths in the content hierarchy.

## Content Hierarchy

```
Learning Journeys (LevelUp) → Courses → Microskillings
```

## Platform

- Microsoft Partner Skilling Hub (skilling-hub.com)
- Official Microsoft Partner training system
- 10+ language support
- Includes AI Skilling Agent for navigation
